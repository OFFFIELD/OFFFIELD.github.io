Hexo Install

  • 极其方便的搭建属于自己的网站

    首先安装Nodejs

  • 地址：https://nodejs.org/

  • 点击页面中央的绿色按钮INSTALL，按引导安装就好了。

新建Github仓库

  • 地址：https://github.com/

  • 添加新仓库OFFFIELD.github.io，OFFFIELD换成自己的用户名

  • 添加SSH-Key

开始安装Hexo

  • 安装命令

    npm install -g hexo

  • 若安装失败，尝试

    sudo npm install -g hexo

  • 初始化Hexo到指定目录

    /*
 * 把指定的文件夹直接拖到init后面            
 * 或用cd命令到目标目录，直接hexo init。
 */
hexo init /Users/ZhiDa/Desktop/Hexo

  • 生成静态页面

    hexo generate

  • 启动本地服务

    /* 地址 http://localhost:4000 */
hexo server

  • 新建文章

    hexo new "MyBlog"

  • 文章格式

    title: postName  /* 标题 */
date: 2013-12-02 15:30:16  /* 时间，会自动生成 */
categories: TEXT  /* 分类 页面右侧导航 */
tags: BLOG  /* 标签 页面右侧导航 */
---

文章摘要，当在首页显示时，会增加read more 按钮
<!--more-->      /* 文章摘要分割线 */
正文

*********************************************************************        
layout: false  /* 如果不需要hexo负责布局 在文章开头添加 */

  • 新建菜单

    hexo new page "about"

  • 清除本地部署

    hexo clean

  • 将新建菜单类型配置到页面

    打开主题文件夹 ~/HEXO/themes/light/_config.yml
记事本打开_config.yml文件
menu:        
      Home: /
      Library: /archives
      About: /about

  • 部署到Git前的配置修改

    /* 编辑hexo主目录下的_config.yml文件，账号换为自己的Github账号 */
# Deployment
## Docs: http://hexo.io/docs/deployment.html
deploy:
      type: git
      repository: https://github.com/offfield/offfield.github.io.git
       branch: master

/* 若失败，请尝试 */
npm install hexo-deployer-git --save

  • 部署到Git

    hexo deploy    
/* 查看结果访问：https://offfield.github.io    */

  • 常用命令缩写

    hexo n = hexo new
hexo g = hexo generate
hexo s = hexo server
hexo d = hexo deploy

  • 复合命令

    hexo deploy -g
hexo server -g

注意

  • Hexo命令必须在init目录下执行
  • 每次修改如果显示不成功，先执行hexo clean 再 hexo generate
  • 每条带冒号命令后面必须有一个空格，否则会报错
