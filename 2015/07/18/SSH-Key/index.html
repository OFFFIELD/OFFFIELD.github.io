Github 添加SSH Key

  1. 打开终端输入

    ssh-keygen -t rsa

  2. 打开Finder前往文件夹

    ~/.ssh/id_rsa.pub

  3. 用记事本打开id_rsa.pub 复制全部内容（SSH Key）
